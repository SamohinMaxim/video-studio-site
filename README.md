# 🎥 Студия видеомонтажа: Онлайн-заказ

Готовый прототип веб-приложения на **Flask** для студии видеомонтажа. Позволяет принимать заказы от клиентов, управлять статусами в админ-панели и автоматически уведомлять клиентов по Email.

## ✨ Возможности проекта

- ✅ **Форма заказа**: Клиент оставляет имя, контакты, описание задачи и ссылку на исходники.
-   *Админ-панель**: Управление заказами с авторизацией по токену.
- 📞 **Быстрые контакты**: Кнопки «Позвонить», «VK» и «Email» прямо в таблице заказов.
- 💌 **Авто-рассылка**: При создании заказа и смене статуса клиенту уходит красивое HTML-письмо.
-    аза данных**: Хранение всех заказов в SQLite (`orders.db`).
-    естовые данные**: Скрипт для быстрой генерации тестовых заказов.
